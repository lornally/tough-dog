# AI史

> 此观点来自于 秋白, 20260302西湖三台阁讲话

ai大模型工具是一个路线, 工具先于生产力发展, 因此编码工具这个领域是最活跃的, 他们沿着一条路发展: 如何更好的使用大模型, 尤其是基于编解码器的大模型

| 产品发布时间  | 工具        | 公司/组织            | 组织成立时间 | 类型          |
| ------------- | ----------- | -------------------- | ------------ | ------------- |
| 2023年3月     | Cursor      | Anysphere            | 2022年       | AI 代码编辑器 |
| 2025年3月6日  | Manus       | 蝴蝶效应 (Monica.im) | 2022年4月    | 通用 AI Agent |
| 2025年6月     | Claude Code | Anthropic            | 2021年       | CLI 编码助手  |
| 2025年6月25日 | Gemini CLI  | Google               | 1998年       | CLI 编码助手  |
| 2026年1月25日 | OpenClaw    | 开源社区             | 2025.11.25*  | Agent 框架    |

\* OpenClaw 首次提交于 2025年11月25日（原名 Clawdbot），2026年1月25日正式发布



**补充说明**：OpenClaw 虽然 2026年1月才正式发布，但其创始人 Peter Steinberger 从 2024年初开始就在密集开发 AI 工具链（18个月开发30+个项目），为 OpenClaw 奠定了基础。

## 时间线观察

**第一波（2023）**：Cursor 最早发布，抓住 GPT-4 后的代码辅助需求

**第二波（2025Q1）**：Manus 作为国内产品爆火，定位为"全球首款通用 AI Agent"，强调端到端任务执行

**第三波（2025Q2）**：Anthropic 和 Google 先后推出 CLI 工具，大厂入局争夺开发者入口

**第四波（2026年初）**：OpenClaw 作为开源框架崛起，与 Cursor、Claude Code 等形成生态联动。项目始于 2025年11月25日 的首次提交（原名 Clawdbot），2026年1月25日正式发布，仅用 90 天就成为 GitHub 史上增长最快的开源项目之一
