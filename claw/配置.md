## 现在需要你操作：

### 第 1 步：打开 Telegram
```bash
open /Applications/Telegram.app
```

### 第 2 步：登录
- 输入手机号
- 输入验证码（会发到手机）

### 第 3 步：创建 Bot（关键步骤）

1. **搜索 BotFather**：在 Telegram 搜索框输入 `@BotFather`
2. **创建新 Bot**：发送 `/newbot`
3. **设置名称**：
   - Bot 名称：`MyKimiClaw`（随意）
   - Bot 用户名：`xxxx_bot`（必须以 _bot 结尾，如 `mykimiclaw_bot`）
4. **复制 Token**：创建成功后会显示：
   ```
   Use this token to access the HTTP API:
   123456789:ABCdefGHIjklMNOpqrSTUvwxyz
   ```
