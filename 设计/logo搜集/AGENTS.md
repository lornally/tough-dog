## 📋 项目概述

**目标**: 建立全网 Logo 库

## 📝 技术规范

### 1. Logo 文件规范

**格式要求**:
- 主文件: SVG (向量格式)
- 预览文件: PNG thumbnail (100x100px)

**文件命名**:
```
主文件: {名称}_logo.svg
预览:   {名称}_logo_thumb.png
```

**存储路径**:
```
logos/by_country/{国家代码}/{名称}_logo.svg
logos/by_country/{国家代码}/thumbs/{名称}_logo_thumb.png
```

### 2. 文件验证流程

每下载一个 Logo，必须:
1. 检查文件大小 (> 1KB)
2. 检查内容 (包含 `<svg` 标签)
3. 生成 PNG thumbnail
4. 如验证失败，删除并记录

### 3. 数据源优先级

1. **worldvectorlogo.com** - 免费 SVG
2. **Wikipedia/Wikimedia** - 品牌 Logo
3. **Clearbit Logo API** - 公司 Logo
4. **各国品牌数据库** - 本地品牌
5. **seeklogo.com** - 补充来源
