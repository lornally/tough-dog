## 🛠️ 技术方案

### Logo 获取策略
1. **API 方式**: Clearbit Logo API (速度快，但可能不是向量格式)
2. **爬虫方式**: worldvectorlogo.com, seeklogo.com (需要解析 HTML)
3. **官方来源**: 公司官网 brand assets 页面 (质量最高但耗时)

### 文件命名规范
```
{公司名称}_{格式}_{变体}.{扩展名}

例如:
- Apple_logo_standard.svg
- Apple_logo_dark.svg
- Apple_logo_inverted.svg
- Apple_icon_mono.svg
```

### 数据记录格式
每个国家的公司列表使用 Markdown 表格：
```markdown
| 公司名称 | 行业 | 规模 | Logo状态 | 本地路径 |
|---------|------|------|---------|---------|
| Apple Inc. | 科技 | 大型 | ✅ | logos/us/apple.svg |
```

## ⚠️ 风险与挑战

1. **数量巨大**: 目标数量远超现有免费数据库 (需多源整合)
2. **版权风险**: 部分 Logo 可能有版权限制
3. **格式一致性**: 不同来源的格式和质量差异大
4. **网站反爬**: 部分网站可能有反爬虫机制
5. **公司基数**: 找到 10,000+ 家/国的公司列表本身就是挑战
